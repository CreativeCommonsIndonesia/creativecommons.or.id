Images

Images can be added in several ways, depending on how you want to present them. By default, images will aligned to the right of the article and have a small box shadow. Generally, you shouldn't change this, but you can if it suits the content of the article.

The different types are the basic image, borderless image, figure, figure group, and carousel.

Steps

  1. The first step for all images is to put the file(s) in the uploads folder of the CCID repo.
  2. Then copy/paste the code for the image type you want to use. If you want to understand more about the classes we use, refer to the Glossary section at the bottom of this document.
  3. Inside the code, replace "example.jpg" with the title of image you want to add, as well as the alt text.

For best results, I recommend pasting the image code above the paragraph it will be appearing next to.

Carousels

A carousel is a slideshow. It's best used when you have a lot of images to show, but I generally recommend against it if you don't know what you're doing.

In the example here, the carousel is aligned right like the other figures, but can be set to full width by deleting float-md-end w-md-50 ms-md-5.

Glossary of classes

w-* sets the width to a specific percentage, so w-100 sets it to 100%, w-50 to 50%, etc.

w-sm-* is a custom class that will set the width to that percentage, but only on non-mobile devices, so it will be 100% on your phone, but not on your desktop. This class is used by default for images, so we don't have overly large images on big screens. w-sm-50 sets it to 50%, w-sm-25 to 25%.

float-sm-end is used to align the item to the right of the page. If you don't want to align it to the right, just delete this class.

m-* describes the spacing of margins, with the letter after the "m" referring to the top, bottom, left, and right of the item. Therefore, the top margin is mt, bottom margin is mb, left margin is ms, and right margin is me. The number describes the size of the margin, with 1 being the smallest and 5 the largest. This means, for example, mt-1 is a small top margin and mt-5 is a large one.

Relatedly, ms-sm-5 means the left margin ("ms") should be large ("5"), but only only non-mobile devices ("sm"). This will give some space to the left of the figure. We use this in conjunction with the image alignment setting, so that there is no ugly margin on phones, and if you delete float-sm-end, you should also delete ms-sm-5.

For additional guidance on these classes, see the Bootstrap documentation: https://getbootstrap.com/docs/5.2/getting-started/introduction/.
